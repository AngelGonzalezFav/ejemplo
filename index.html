Edicion de paginas web utilizando Dreamweaver

 

Vamos a aprender a utilizar un programa que permita diseñar y gestionar los sitios web de manera profesional, dinamica y sencilla. El editor de Dreamweaver le permitira explorar las diferentes opciones de creacion y diseño sin compliacarte con codigos y etiquetas.

Esperamos que puedas familiarizarte con el entorno de trabajo y que tus practicas esten a la altura de lo que se pide.

Dreamweaver es facil. Vamos a comporbarlo en las proximas semanas.
